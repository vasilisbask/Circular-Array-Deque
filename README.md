# Circular Array Deque

This code is part of an exercise for the Data Structures course at Harokopio
University of Athens, Dept. of Informatics and Telematics. It implements a deque using a circular array.

## Usage

Compile using

```
mvn compile
```

Create a jar using

```
mvn package
```

Run main using

```
java -cp .\target\DeQueue-1.0-SNAPSHOT.jar com.mycompany.dequeue.App
```

Run unit tests using

```
mvn test
```
